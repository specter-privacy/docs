## Core tests

```bash
cd SPECTER/specter
cargo test -p specter-core -p specter-crypto -p specter-stealth -p specter-registry -p specter-scanner
```

## API tests

```bash
cargo test -p specter-api
```

## Benchmarks

```bash
cargo bench -p specter-crypto
cargo run --bin specter -- bench --count 100000
```
