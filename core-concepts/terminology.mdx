---
title: "Terminology"
description: "Key SPECTER protocol terms"
---

| Term | Meaning |
| --- | --- |
| Meta-address | Public bundle of `spending_pk` + `viewing_pk` published via name service |
| Spending keypair | Keypair used to derive spend-capable stealth keys |
| Viewing keypair | Keypair used to discover incoming payments |
| Announcement | Registry record carrying `ephemeral_key`, `view_tag`, metadata |
| View tag | 1-byte filter value derived from shared secret |
| Stealth address | One-time address derived for a single payment |
| Shared secret | 32-byte secret from ML-KEM encapsulation/decapsulation |
| Channel ID | Optional 32-byte identifier used by Yellow channel flows |
