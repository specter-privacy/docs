## Choose your integration setup mode

<Tabs>
  <Tab title="Frontend only">
    No installation is needed. Use the Launch app button and follow `/explore/frontend-experience`.
  </Tab>
  <Tab title="API integration">
    Prerequisites: Rust stable toolchain, `PINATA_GATEWAY_URL`, and `PINATA_GATEWAY_TOKEN`.

```bash
git clone https://github.com/pranshurastogi/SPECTER.git
cd SPECTER/specter
cp .env.example .env
# fill gateway variables
cargo run --bin specter -- serve --port 3001
```

  </Tab>
  <Tab title="Frontend + local backend">

```bash
cd SPECTER/SPECTER-web
cp .env.example .env
npm install
npm run dev
```

    Use this mode when you are iterating on UX while validating local API behavior.
  </Tab>
</Tabs>

## Verify setup quickly

<Steps>
  <Step title="Health check">

```bash
curl -s http://localhost:3001/health | jq .
```

  </Step>
  <Step title="Key generation check">

```bash
curl -s -X POST http://localhost:3001/api/v1/keys/generate | jq .
```

  </Step>
  <Step title="Move to full integration loop">
    Continue to `/getting-started/quickstart`.
  </Step>
</Steps>
