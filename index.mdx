## Post-Quantum Privacy for Ethereum and Sui

SPECTER is the first production-ready protocol for **quantum-resistant private payments**. It enables users to receive assets without revealing their primary address, protecting them from surveillance and future quantum threats.

## Core Capabilities

![Quantum Security](/images/specter/quantum-shield.png)

SPECTER is built for speed, security, and developer experience.

<CardGroup cols={2}>
  <Card title="Quantum Resistance" icon="shield-halved">
    Secured by **ML-KEM-768** (Kyber), the NIST standard for post-quantum cryptography.
  </Card>
  <Card title="Multi-Chain Support" icon="link">
    Native support for **Ethereum (EVM)** and **Sui**, with more chains coming soon.
  </Card>
  <Card title="Stealth Addressing" icon="user-secret">
    Derive unique, unlinkable addresses for every transaction.
  </Card>
  <Card title="Decentralized Registry" icon="globe">
    Announce payments via 1-byte view tags for efficient scanning.
  </Card>
</CardGroup>
