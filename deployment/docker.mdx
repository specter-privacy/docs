---
title: "Docker deployment"
description: "Build and run the SPECTER API container"
---

## Build image

```bash
cd SPECTER/specter
docker build -t specter-api:local .
```

## Run container

```bash
docker run --rm -p 3001:3001 --env-file .env specter-api:local
```

## Validate

```bash
curl -s http://localhost:3001/health | jq .
```

<Info>
The Docker runtime command starts `specter serve` and exposes port `3001`.
</Info>
