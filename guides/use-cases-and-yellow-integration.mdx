## Yellow integration evidence (source-backed)

- API routes expose the Yellow namespace in `specter/specter-api/src/routes.rs`.
- Endpoint handler behavior is implemented in `specter/specter-api/src/handlers.rs` (`yellow_create_channel`, `yellow_discover_channels`, `yellow_fund_channel`, `yellow_close_channel`, `yellow_transfer`, `yellow_config`).
- Default Yellow runtime configuration is initialized in `specter/specter-api/src/state.rs`.
- Frontend Yellow flow is implemented in `SPECTER-web/src/pages/YellowPage.tsx`.
- Frontend use case presentation is implemented in `SPECTER-web/src/pages/UseCasesPage.tsx`.
